# 🌴 Siam Delights

**Siam Delights** brings the vibrant flavors of Thailand into Minecraft.
Grow tropical fruits, cook authentic Thai dishes, and expand your culinary adventure with full support for Farmer’s Delight.

---

## ✨ Features

* 🍍 Tropical fruits: mango, coconut, papaya, banana, and more
* 🍜 Thai dishes like Pad Thai, Tom Yum, and Mango Sticky Rice
* 🍳 Integration with Farmer’s Delight cooking system
* 🌿 Vanilla-friendly balance and progression

---

## ⚙️ Requirements

* Fabric Loader
* Fabric API
* (Optional) Farmer’s Delight Refabricated

---

## 🚀 Installation

1. Install Fabric Loader
2. Download and install Fabric API
3. (Optional) Install Farmer’s Delight Refabricated
4. Place the mod `.jar` file into your `mods` folder

---

## 🧑‍🍳 Recipes & Gameplay

Siam Delights integrates with Farmer’s Delight:

* Cooking pot recipes
* Ingredient compatibility
* Expanded food crafting system
